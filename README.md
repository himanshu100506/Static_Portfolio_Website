# Project Overview

## Static Portfolio Website

The **Static Portfolio Website** is a simple, responsive, and visually clean personal portfolio built using **HTML and CSS**.  
It is designed to present personal information such as:

- Introduction  
- Skills  
- Projects  
- Contact details  

This website is ideal for students, beginners, or anyone wanting a lightweight personal portfolio for college practicals or GitHub hosting.
